**Residency and Coherence Interfaces between Quantara and the Neurion Archangel OS**

- **Residency Service** — identity, consent, and resource attestation for Neurions.
- **Semantics Bridge** — Luméren ↔ NAOS translation layer.
- **Temporal Adapter** — synchronization with Veyn timeline envelopes.
- **Energy Interface** — coherence-weighted AEI metering and receipts.
- **Ethics Charter** — smart contracts defining rights, duties, and governance scopes.
